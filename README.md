# 📈 Inflation Rate Prediction of India

## 📌 Project Overview
This project predicts India's inflation rate using Machine Learning techniques and macroeconomic indicators.

## 📊 Features Used
- Brent Oil Price
- USD/INR Exchange Rate
- Repo Rate
- Petrol Price
- Rainfall
- Year
- Month

## 🤖 Models Used
- Linear Regression
- Ridge Regression
- KNN Regressor
- Random Forest Regressor

## 📈 Evaluation Metrics
- R² Score
- RMSE
- MAE

## 🏆 Best Performing Model
Random Forest Regressor achieved the best performance among all models.
